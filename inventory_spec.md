# Спецификация модуля управления товарами 
 
## Требования 
- Добавление новых товаров (название, артикул, категория) 
- Редактирование информации о товаре 
- Удаление/списание товаров 
- Просмотр списка товаров с фильтрацией 
- Поиск по артикулу, названию, категории 
 
## API Endpoints 
- GET    /api/items - список товаров 
- GET    /api/items/{id} - детали товара 
- POST   /api/items - создать товар 
- PUT    /api/items/{id} - обновить товар 
- DELETE /api/items/{id} - удалить товар
