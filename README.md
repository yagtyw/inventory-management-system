\# Система управления инвентаризацией



\## Описание проекта

Система для автоматизации складского учета автозапчастей. 

Позволяет отслеживать движение товаров, проводить инвентаризацию, 

управлять остатками и синхронизироваться с интернет-магазином.



\## Основные функции

\- Управление товарами (CRUD операции)

\- Сканирование штрих-кодов и QR-кодов

\- Инвентаризация с подсчетом расхождений

\- Уведомления о низких остатках и сроках годности

\- Отчеты и аналитика

\- Офлайн-режим с синхронизацией

\- Интеграция с интернет-магазином



\## Технологии

\- Backend: Python/FastAPI

\- Frontend: React

\- Mobile: React Native

\- Database: PostgreSQL
